# 项目概览

## 任务所·Flow v1.7

**Slogan**: 用对话，开工；用流程，收工。AI写代码新范式

### 项目定位
企业级AI任务协作与知识管理系统

### 核心能力
1. **AI团队体系** - 4角色(架构师/工程师/代码管家/SRE)完整协作
2. **知识库管理** - 12表知识图谱，问题-解决方案-决策关联
3. **任务协作** - 依赖分析、状态追踪、进度监控
4. **架构师即插即用** - 任何项目10分钟生成架构报告

### 技术架构
- **架构**: Monorepo (8个顶层目录)
- **后端**: FastAPI + Python
- **前端**: HTML/CSS/JS (工业美学)
- **数据库**: SQLite (12表)
- **AI**: Claude 3.5 Sonnet
