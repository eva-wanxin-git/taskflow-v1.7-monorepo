### 方法1: 清除浏览器缓存（推荐）

```
1. Ctrl+Shift+Del
2. 选择"缓存的图片和文件"
3. 时间范围：全部
4. 点击"清除数据"
5. 刷新Dashboard（F5）
```

### 方法2: 使用Dashboard清除按钮

```
1. 打开 http://localhost:8877
2. 找到"清除缓存"按钮（缓存版本那一行）
3. 点击 "🔄 清除缓存"
4. 等待2秒自动刷新
```

### 方法3: 换端口（最可靠）

```
1. 关闭当前Dashboard
2. 端口改为8878
3. 重新访问 http://localhost:8878
```
