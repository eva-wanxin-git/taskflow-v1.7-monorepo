## 🎯 如何使用

### 步骤1: 找到进行中的任务

Dashboard上查看"进行中"状态的任务，例如：
- INTEGRATE-003

### 步骤2: 点击"🔄 重新派发"按钮

会看到确认对话框：
```
确定要重新派发任务 INTEGRATE-003 吗？

这会：
1. 复制完整的任务提示词
2. 可选：重置任务状态为pending

建议：发给新的执行者，避免重复工作
```

### 步骤3: 确认并复制

- 点击"确定" → 提示词自动复制到剪贴板
- 看到成功提示：✅ 重新派发提示词已复制！

### 步骤4: 是否重置状态

会询问：
```
是否将任务状态重置为pending？

建议：如果已确认原执行者放弃，点击"确定"重置
```

- 点击"确定" → 状态重置为pending
- 点击"取消" → 状态保持in_progress

### 步骤5: 派发给新执行者

```markdown
@新Cursor

（粘贴刚才复制的内容）
```
