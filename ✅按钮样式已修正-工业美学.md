## 📐 工业美学原则

### 1. 极简
```
去掉: "一键"、"完成"等啰嗦词
保留: 核心功能词
```

### 2. 统一
```
所有按钮样式一致
只通过文字和颜色区分功能
```

### 3. 符号
```
使用几何符号: ▸ ↻ ◉ ◆
不用emoji: 🔄 📋 📄
```

### 4. 颜色
```
默认: 黑色文字 + 浅灰边框
强调: 赭红色文字（重新派发）
Hover: 黑色边框
```
